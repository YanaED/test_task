# Оптимизация стратегий взыскания долгов

- `Тестовое задание_Егорова.ipynb` - Jupyter Notebook с полным кодом и пояснениями
- `test_task.py` - Python скрипт для запуска дашборда

Для запуска необходимо установить библиотеки, если они не установлены:
```bash
pip install dash pandas numpy plotly
```

Запуск кода:
1. Через Jupyter Notebook - открыть `Тестовое задание_Егорова.ipynb` и выполнить все ячейки.
2. Через  терминал (дашборд будет доступен по адресу [http://127.0.0.1:8050](http://127.0.0.1:8050)
   ```bash
   python test_task.py
   ```
